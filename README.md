# Приложение для выбора фильма для совместного просмотра

Веб-приложение для помощи в выборе фильма для совместного просмотра.  
Пользователи присоединяются к общей сессии и выбирают фильмы из заранее заданной подборки.  
Фильмы, понравившиеся всем участникам сессии, формируют итоговый список совпадений.

Приложение по логике работы аналогично сервисам типа «chto?» и подобным.

---

## Основные возможности

- Создание сессии для совместного выбора фильма
- Присоединение нескольких пользователей к одной сессии
- Выбор подборки фильмов лидером сессии
- Просмотр списка фильмов из выбранной подборки
- Индивидуальный выбор понравившихся фильмов
- Формирование списка фильмов, совпавших у всех участников сессии
